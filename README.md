# ⏱️ Cronômetro em C#

Este projeto é um **cronômetro desenvolvido em C# (Console Application)**, criado com o objetivo de praticar lógica de programação, manipulação de strings e conversão de tipos.

O usuário pode definir o tempo do cronômetro de forma flexível, utilizando **segundos (`s`) ou minutos (`m`)**.

---

## 🚀 Funcionalidades

✅ Cronometragem em tempo real  
✅ Entrada de tempo definida pelo usuário  
✅ Suporte para segundos (`s`) e minutos (`m`)  
✅ Atualização do tempo no console  
✅ Mensagem ao finalizar o cronômetro  

---

- **C#**

- **Aplicação de Console**
- **Visual Studio Code**

## ⌨️ Como funciona a entrada de dados

O usuário deve informar o tempo no seguinte formato:

- `10s` → 10 segundos  
- `1m` → 1 minuto  
- `5m` → 5 minutos  

O programa separa:
- O **valor numérico**
- O **tipo de tempo** (`s` ou `m`)

E converte automaticamente para segundos antes de iniciar o cronômetro.
